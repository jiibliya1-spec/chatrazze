# Chatraze Messaging Application

Chatraze is a messaging application that allows users to communicate in real-time. This project aims to provide a robust and user-friendly platform for messaging.

## Features
- Real-time messaging
- User authentication
- Group chats
- File sharing

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jiibliya1-spec/Chatraze.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Chatraze
   ```
3. Install dependencies for backend and frontend.

## Usage
- Start the backend server.
- Start the frontend application.
- Enjoy chatting!
